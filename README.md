# reserving-paper

Companion code for the paper:
**"When Does Machine Learning Beat Traditional Actuarial Methods for Loss Reserve Estimation? A Systematic Comparison"**

Submitted to Variance: Journal of the Casualty Actuarial Society.

## Data

CAS Schedule P loss reserve data — download from [casact.org](https://www.casact.org/publications-research/research/research-resources/loss-reserving-data-pulled-naic-schedule-p) and place CSVs in `data/raw/`.

## Setup

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
```
